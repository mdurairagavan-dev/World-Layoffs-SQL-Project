# World Layoffs - SQL Project

## Tools Used
- MySQL / MySQL Workbench

## Dataset
- Source: Kaggle - Layoffs 2022
- 2,361 rows of global layoffs data

## Project Steps

**1. Data Cleaning**
- Removed duplicates
- Standardized industry and country names
- Fixed date format
- Removed useless null rows

**2. Exploratory Data Analysis (EDA)**
- Which companies had the most layoffs?
- Which country and industry was hit hardest?
- Layoff trends by year
- Rolling total of layoffs per month
